Problem

Bill Gates loves Cipher.

Attachement: http://ctf.nullcon.net/trivia/trivia3.png

Solution

This one took some time to figure out, mostly because the description is so vague. But using reverse image search for the cipher characters and “Bill” as keyword (Gates was a red herring), we managed to find Bill’s Cipher.

With the substitution table, it was very easy to get the proper string.
